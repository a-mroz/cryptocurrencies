# Cryptocurrencies

Demo project for Gamedia.


# Building
Requires Java 14. Build with gradle:
```shell script
./gradlew build
```

# Running
After building can be started with:
```shell script
java -jar build/libs/cryptocurrencies-0.0.1-SNAPSHOT.jar 
```

or, using Gradle:
```shell script
./gradlew bootRun
```
